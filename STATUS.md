## Экраны (спецификация кратко)

- **Дашборд** — DnD-сетка виджетов: воронка конверсии, NPS/поддержка, скорость команды, инциденты, уведомления.
- **Ассистент** — 3 колонки: список диалогов / лента сообщений / контекст. Команды `/`, упоминания `@`.
- **Агенты** — вкладки: Готовые / Мои / Конструктор (триггер → условие → действие).
- **ИИ-сервисы** — каталог сторонних инструментов с фильтрами, рейтингом, корп-подпиской.
- **Задачи** — Канбан / Список / Бэклог (авто-скоринг ICE/RICE+compliance) / Таймлайн (SVG Gantt).
- **Комнаты** — типы: Планирование спринта / Ретро / Груминг. Список → детальная комната.
- **База знаний** — граф артефактов (задача, PR, метрика, Confluence, исследование) + поиск.
- **Уведомления** — выпадающая панель из хедера + `/notifications`.
- **Профиль** — вкладки: Уведомления, Интеграции, Персонализация, Агенты/сервисы, Команды.

## Ключевые решения

| Решение | Детали |
|---------|--------|
| Стек | Vite + React + TypeScript strict, antd v5 тёмная тема |
| Данные | Фикстуры через `data/api/`; реальные интеграции — фаза 4 |
| ИИ | Express API `scripts/dev-api-server.ts` на порту 3001; SSE стриминг через Anthropic SDK |
| Авторизация | HMAC-SHA256 токены (Web Crypto); логин/пароль только на сервере; `arsendsgn/481516` |
| Хостинг | VPS Timeweb (72.56.34.107), nginx + PM2 + certbot |
| CI/CD | GitHub Actions: build → SCP dist/ → SSH restart PM2; push → main |
| Прокси | Cloudflare Worker `anthropic-proxy.arackelian.workers.dev`; репо `arsen2317/anthropic-proxy` |
| Роутинг воронки | `users` = квартальный итог; дневная история суммируется ≈ users |
